Aplicando Regras

Paragrafo 1 - @media: um grupo de regras condicional que é aplicado ao conteúdo caso o dispositivo de exibição atenda com a condição de mídia definida (media query).

@charset: define o conjunto de caracteres (codificação) usado pelo CSS.

Colocando img

O CSS tem diversas regras, que são denominadas At-Rules, que são declarações com instruções de como o CSS irá se comportar

At-rules são iniciadas com o sinal de arroba (@), seguido por um identificador e terminando com um ponto-e-vírgula (;).
